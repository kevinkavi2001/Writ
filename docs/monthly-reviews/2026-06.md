- Phase 6j (methodology integration verification) shipped via the multi-part change in commit-pending: `node_types` opt-in on `QueryRequest`, methodology-companion query in `writ-rag-inject.sh`, and live Neo4j migration of the methodology corpus.

Cumulative cost now visible via `event: always_on_inject` friction events.

Live Neo4j ingestion was missing until this session ran `scripts/migrate.py --methodology-dir bible/methodology` (added 50 new-type + 10 methodology Rule nodes; 120 edges).
